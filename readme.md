## Overview

Open "Developer Command Prompt for VS 2019"
(or run command `"C:\Program Files (x86)\Microsoft Visual Studio\2019\Enterprise\Common7\Tools\VsDevCmd.bat"`).

```
cd <project dir>
cmake . -B vsproj
cd vsproj
```

Open `test_dll_load.sln`.

Compile.
